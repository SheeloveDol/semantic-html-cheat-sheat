Semantic HTML Cheatsheet

Semantic HTML makes our code more readable for both ourselves and other developers. It can also make the content of our page more accessible to users who use readers to browse the web. Below is a table with some of the most commonly used semantic html tags and a brief description of when and how to use them.

Tag Name Description
<header> Header Typically contains a website's top navigation and menu
<main> Main Primary content of the webpage
<footer> Footer Typically contains copyright and contact info found at the bottom of a page
<title> Title Shows up in the browser tab and search results. Very Important
<article> Article An independent piece of content that make sense on its own
<section> Section A group in a series of related content pieces
<nav> Nav Contains a group of navigation links
<figure> Figure Embeds annoted images, illustrations, photos, code
